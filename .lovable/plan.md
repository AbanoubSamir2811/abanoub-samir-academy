## التصميم والحركة

- ثيم داكن: خلفية `#0a1024` تقريبًا، تدرّجات أزرق `#1e90ff` → بنفسجي `#7c3aed`
- خط عربي حديث (Cairo أو Tajawal)
- Framer Motion لجميع الحركات: ظهور تدريجي عند التمرير، hover scale، انتقالات صفحات سلسة
- بطاقات بحدود متوهجة وزجاج شفاف
- حركات هادئة وراقية (مدة 300–600ms)، ليست مبالغًا فيها

## التفاصيل التقنية

**التقنيات**: TanStack Start + React + Tailwind v4 + Framer Motion + react-hook-form + zod + Lovable Cloud (Supabase).

**قاعدة البيانات** (Lovable Cloud):
- جدول `bookings`: id, full_name, phone, academic_year (enum), study_type (enum), notes, created_at
- جدول `user_roles` + enum `app_role` + دالة `has_role` (نمط الأدوار الآمن)
- RLS:
  - `bookings`: INSERT مسموح للجميع (للزوّار)، SELECT/DELETE للمسؤول فقط عبر `has_role(auth.uid(),'admin')`
  - `user_roles`: قراءة ذاتية فقط
